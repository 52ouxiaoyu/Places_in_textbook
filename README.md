# 课本里的中国：100个真实地点 × 100个有温度的故事提纲

> **说明**  
> - 所有地点均出自部编版/人教版小学或初中语文、历史、地理教材  
> - 每个故事基于正史、地方志、考古报告、人物文集等可靠史料  
> - 避开“孟姜女哭长城”“白蛇传”等过度传播的传说  
> - 每则包含：**标题｜核心事件｜情感基调｜价值点｜史料来源提示**
